# [TEAM O] SODI

이 Repository는 "SODI" 의 네이티브 앱을 담당하는 Repository 입니다.

다음과 같은 기능이 포함되어 있습니다.

- 로그인, 회원가입
- 피드 리스트
- 프로필
- 옷장
